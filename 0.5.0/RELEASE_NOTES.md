# CineRename 0.5.0

CineRename 0.5.0 is the first public release and the most stable build available today. It is ready for day-to-day library cleanup, bulk renaming and server workflows; the 0.5.x version number simply leaves room for final user feedback before the 1.0 milestone.

## Highlights

- Rename and organize movies, series and anime with a safe preview before touching files.
- Ready-made Plex, Jellyfin and Kodi naming profiles, including media folders and ID-aware formats.
- Anime support with absolute episode numbering, AniList/Kitsu lookup and localized episode titles through TheTVDB when available.
- Dry-run exports, rename history and undo/rollback for safer bulk operations.
- Subtitle search with OpenSubtitles, language selection, preview, conversion and timing adjustment tools.
- Library audit, NFO generation, poster/fanart download and metadata sidecars for media servers.
- Watch folders, cautious automatic pipeline, qBittorrent / Transmission / JDownloader integrations and Pre-Arr workflows.
- Headless CLI, WebUI, TUI, scheduler, NAS tarballs and Docker archives for server use.

## Notes Before Installing

- Windows and macOS builds are currently unsigned. Windows SmartScreen and macOS Gatekeeper may show a warning on first launch.
- Direct auto-update metadata is published for desktop updater builds. Store/package-manager builds will use their own update channels later.
- Store packages such as Microsoft Store, Snap, Flatpak, AUR, Homebrew, winget and Scoop are planned but not published yet.
- Docker is published as downloadable image archives in this release; registry publishing can be added later.
