# CineRename Releases

CineRename 0.5.0 is being prepared for its first public release.

Downloads are temporarily hidden while the release artifacts are regenerated with the latest documentation and packaging metadata. The next published `v0.5.0` release will restore direct download links for Windows, macOS, Linux desktop, NAS/headless archives, and Docker image archives.
